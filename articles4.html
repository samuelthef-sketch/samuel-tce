    Clap Light Bulb

    This project is a sound-activated light control system that turns a light bulb ON or OFF when you clap your hands. It uses a sound sensor (microphone module) to detect the sound of a clap and a relay module to control the bulb.

    Code:

    
#define sensorPin 7
#define relayPin 8

// Variable to store the time when last event happened
unsigned long lastEvent = 0;
boolean relayState = false;  // Variable to store the state of relay

void setup() {
  pinMode(relayPin, OUTPUT);  // Set relay pin as an OUTPUT pin
  pinMode(sensorPin, INPUT);  // Set sensor pin as an INPUT
}

void loop() {
  // Read Sound sensor
  int sensorData = digitalRead(sensorPin);

  // If pin goes LOW, sound is detected
  if (sensorData == LOW) {

    // If 25ms have passed since last LOW state, it means that
    // the clap is detected and not due to any spurious sounds
    if (millis() - lastEvent > 25) {
      //toggle relay and set the output
      relayState = !relayState;
      digitalWrite(relayPin, relayState ? HIGH : LOW);
    }

    // Remember when last event happened
    lastEvent = millis();
  }
}

    Hmmmmmm

    • I use a relay module rated for my mains voltage and follow safety precautions when wiring mains loads.
    • Adjust timing thresholds (50 ms / 400 ms) if my clap timings differ.
