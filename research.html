      Smart Pole Monitoring System (SPMS)

      How it works and the idea behind the project

      The Smart Pole Monitoring System (SPMS) is designed to detect problems in electrical poles and automatically send signals to a consumer computer App. Once a fault is detected, the system updates a map-based application, similar to Google Maps, so that electricity consumers and service teams can see which poles have issues. This allows for safer, faster, and more efficient maintenance.

      The system can also detect the angle of the pole using an accelerometer or gyroscope. If the pole tilts between 50 to 130 degrees, it is considered at risk or fallen, and the system immediately sends a signal to the computer to update the map and alert the service team.

      The idea for this project came after Typhoon Uwan hit on November 10, causing widespread power blackouts. While I was going through a memo posted on a local Facebook page for electricity consumers, I noticed that the team mentioned they would still manually check all the poles for problems. That made me think: it takes a long time if they manually check every pole, so what if we create an SPMS that touches the pole, detects problems automatically, and sends data to a computer? Coupled with a maps application, consumers can easily see which poles are affected, improving safety and reducing response time.
